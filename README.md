Create a new C# console application:
```dotnet new console -n HelloWorld```

1. using System;: Imports the System namespace, which provides essential classes like Console that help with input/output.
2. class Program: Defines a class named Program. In C#, everything goes inside classes.
3. static void Main(): This is the entry point of every C# application. The Main method is called when you run your program.
4. Console.WriteLine("Hello, World!");: Outputs the text Hello, World! to the terminal.


`dotnet run`
